1. INTRODUCTION
 ## 1.1 Purpose
 PLANWISE is a AI-powered study planner designed to help students organize their study schedules, manage subjects ,prepare for exams ,and improve productivity through intelligent planning and progress tracking.

 The purpose of this project is to provide students with a personlized learning assistant that automatically create study plans based on the student's available time,subjects ,goals, and deadlines.

 ## 1.2  SCOPE
 PLANWISE will allow the users to:
 - Create an account
 - Securly login
 - Add subjects
 - Add exams
 - Set study goals
 - Generate AI based study plans
 - Track study progress
 - View analytics
 - Receive intelligent study recomendations 
 THE APPLICATION IS INTENDED TO WORK ON  BOTH DESKTOP AND MOBILE WEB BROWSERS

2. PROBLEM STATEMENT

 Many strugle to manage multiple subjects ,assingments. examinations and revision schedules.
Most existing study planners are static and require students to mnuallly organize their tasks.
PLANWISE solves this problem by using Artificial Intelligence to generate personalized study schedules that adapt according to the student's progress , deadlines, and available study time .
