# User Stories

# PLANWISE
AI-Powered Study Planner

---

## Epic 1: User Authentication

### User Story 1

As a student,
I want to create an account,
so that I can securely save my study plans.

---

### User Story 2

As a student,
I want to log into my account,
so that I can access my personalized dashboard.

---

### User Story 3

As a student,
I want to edit my profile,
so that my information stays updated.

---

## Epic 2: Subject Management

### User Story 4

As a student,
I want to add subjects,
so that I can organize my studies.

---

### User Story 5

As a student,
I want to edit or remove subjects,
so that my planner stays accurate.

---

## Epic 3: Exam Management

### User Story 6

As a student,
I want to add upcoming exams,
so that the planner can prioritize my schedule.

---

### User Story 7

As a student,
I want to update exam dates,
so that my study plan adjusts automatically.

---

## Epic 4: AI Study Planner

### User Story 8

As a student,
I want AI to generate a personalized study timetable,
so that I don't have to create one manually.

---

### User Story 9

As a student,
I want AI to adjust my schedule if I miss a study session,
so that I stay on track.

---

### User Story 10

As a student,
I want AI to recommend revision sessions,
so that I remember concepts better.

---

## Epic 5: Progress Tracking

### User Story 11

As a student,
I want to mark tasks as completed,
so that I can monitor my progress.

---

### User Story 12

As a student,
I want to see my daily and weekly analytics,
so that I know how consistently I am studying.

---

### User Story 13

As a student,
I want to maintain study streaks,
so that I stay motivated.

---

## Epic 6: AI Assistant

### User Story 14

As a student,
I want to ask AI study-related questions,
so that I can quickly understand difficult topics.

---

### User Story 15

As a student,
I want AI to motivate me when I lose consistency,
so that I continue studying.

---
